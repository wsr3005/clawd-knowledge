# AI / Agent 战略情报｜2026-07-20

**一句话结论：** 今天真正值得动手的不是追逐新模型传闻，而是抓住三项已落地的工作流变化：云端 Agent、代码安全扫描、Claude Code 临时增容。

## 1. ChatGPT Work 把 Agent 从“开着电脑跑”改成云端异步执行

【事实】**已证实。** OpenAI 联合创始人 Greg Brockman 明确表示，ChatGPT Work 在云端运行，可从手机使用，笔记本合盖后仍能继续工作；OpenAI 文档同时显示 Work mode 可在 Web 与移动端使用。这里不沿用二手来源关于套餐覆盖面的说法。

【影响】这改变的是执行拓扑，而不只是多了一个入口：适合把资料汇总、文档生产、跨插件业务任务迁到云端队列；需要本机文件、私有环境或精细终端控制的任务仍留在 Hermes / Codex 本地链路。移动端由此从“查看结果”升级为“发起和续接长任务”。

【动作】把任务按依赖切成两类：无本机依赖、可异步验收的任务优先试跑 ChatGPT Work；涉及本地仓库、密钥或不可逆写操作的任务继续使用本地 Agent，并保留人工批准点。先用一个真实的资料整理或文档交付任务做端到端验收，不因产品宣传直接迁移主流程。

【证据】[Greg Brockman 第一方说明](https://x.com/gdb/status/2078922461660533120)；[OpenAI Work mode 管理文档](https://developers.openai.com/codex/enterprise/work-admin-faq)

【可信度/待核验点】高。云端、移动端与合盖继续运行已由 OpenAI 核心人员说明；昱桥账号的具体可用范围、额度和插件权限仍需以产品界面及工作区设置为准。

## 2. Codex Security 插件已可把安全扫描接入仓库验收

【事实】**已证实。** OpenAI 已提供 Codex Security 插件：可对本地 Git 仓库、单个文件夹或无 Git 历史的代码库执行首次只读标准扫描；结果包含发现、覆盖范围与证据。官方变更记录还支持将结果导出为 JSON、CSV 或 SARIF，并在审批后把已验证问题写入 GitHub、Jira 或 Linear。

【影响】这补上了 Agent 编码流程最危险的一段：代码生成速度越来越快，但安全审查不能继续靠“模型顺手看看”。它可以成为发布前的独立验收层，并以结构化结果接入问题追踪，而不是把扫描结论埋在聊天记录里。

【动作】把 Codex Security 标准扫描加入下一次真实项目发布前检查：先只读扫描并人工复核高置信度问题，再决定修复；只有标准扫描与构建验证稳定后才跑更慢的 deep scan。涉及创建 Issue 或安全公告时继续保留显式批准，不让插件自动外发。

【证据】[OpenAI：Codex Security 插件快速入门](https://developers.openai.com/codex/security/plugin)；[OpenAI：插件变更记录](https://developers.openai.com/codex/security/plugin/changelog)；[OpenAI 官方 X 操作说明](https://x.com/OpenAI/status/2078243670265041038)

【可信度/待核验点】高。能力与流程均来自官方文档；具体账号可见性、扫描额度及扫描质量尚未在昱桥的真实仓库上验证。

## 3. Claude Code 周额度临时提高 50%，延续至 8 月 19 日

【事实】**已证实。** Anthropic 官方开发者账号宣布：Claude Code 的周使用上限继续提高 50%，覆盖 Pro、Max、Team 与按席位计费的 Enterprise 用户，延续至 2026 年 8 月 19 日。其引用的 Claude 官方公告还称，自 7 月 20 日起，Fable 5 纳入 Max 与 Team Premium，额度占总限额的 50%；Pro 与 Team Standard 继续通过 usage credits 使用。

【影响】短期内 Claude Code 的可用容量和 Fable 5 的套餐边界已经变化，旧的“额度不够所以立即加购或迁移”判断需要重算。它也提供了一个明确的一个月窗口，可用真实高难任务比较 Claude Code 与 Codex，而不是依据零散 benchmark。

【动作】8 月 19 日前优先把架构审查、复杂重构和长程任务放进 Claude Code 实测，并记录每类任务的完成率、人工返工与额度消耗；不要把临时增容当作永久基线，也不要仅因本次活动立即升级套餐。

【证据】[Anthropic / ClaudeDevs 官方公告](https://x.com/ClaudeDevs/status/2078511173759324328)；[Claude 官方套餐变更原帖](https://x.com/claudeai/status/2078302415804379218)

【可信度/待核验点】高。期限、覆盖计划与增幅来自官方账号；昱桥账号实际显示的剩余额度、地区可用性及 credits 入账仍需以账户页为准。
